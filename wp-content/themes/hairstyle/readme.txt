== Description ==

Hairstyle theme is responsive hair salon WordPress theme. Hairstyle WordPress Theme Designed for spa, salon, beauty care, medical business, massage centre, beauty centre and eCommerce websites. Beautifuly designed theme comes with easy to manage theme options in customizer.  This theme is fully responsive working well with all the resolutions.  Check demo at: http://flythemesdemo.net/hairstyle/
